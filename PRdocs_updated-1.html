ARCHITECTURE

main.py · startup flow # ── Startup Sequence ────────────────────────────────────────────── async def boot(): mk_log() # RotatingFileHandler (3MB × 5 backups) scan_plugins() # Dynamic importlib plugin loader await app.start() # Pyrogram/Kurigram client await init_db() # Motor MongoDB connect (pool 20–200) await ping_db() # Health check await WORLD_CYCLE.start() # Day/Night background service await send_gc(report) # Boot report to LOG_GC await idle()
config.py · env vars class Config: API_ID # Telegram API ID (required) API_HASH # Telegram API hash (required) BOT_TOKEN # Bot token (required) MONGO_URI # MongoDB URI (required) SUDO_USERS # Comma-sep int IDs LOG_GC # Log group chat ID WORKERS # default 64 DAY_NIGHT_INTERVAL_SEC # default 900
database/db.py · collections usersdb = db.users itemsdb = db.items questsdb = db.quests clansdb = db.clans nationsdb = db.nations marketdb = db.market worlddb = db.world_state
⚡
ASYNC FIRST

All DB calls, API requests and heavy math are awaited — never blocking the Pyrogram event loop. Motor driver throughout.

KURIGRAM · MOTOR
🔌
PLUGIN LOADER

scan_plugins() uses importlib to dynamically load all *.py files in bot/plugins/. Failed plugins log but never crash boot.

IMPORTLIB · SCAN
🗄️
MONGODB

7 collections: users, items, quests, clans, nations, market, world_state. Pool 20–200. DNS forced to 8.8.8.8 for VPS stability.

MOTOR · ASYNC
🌗
WORLD CYCLE

Background asyncio task ticks day/night phase at configurable intervals (DAY_NIGHT_INTERVAL_SEC). Survives tick failures gracefully.

WORLD_CYCLE_SERVICE

CORE GAME SYSTEMS

🎭
CHARACTER PATHS

Three paths at creation: Warrior 🛡️, Rogue 🗡️, Mage 🔮. Each unlocks path-specific skills from SKILLS_CATALOGUE with unique synergies and stat scaling.

MECHANICS.PY · START.PY
📦
INVENTORY SYSTEM

Item, Enchantment, ItemSkin, Inventory dataclasses (slots=True). 4-slot chests: weapons, armoury, potions, tools. 60-item cap. Enchants stack additively.

INVENTORY.PY
💰
DEEP ECONOMY

Gold/Gems/Bank wallet. Gold sinks via durability, market tax, healing costs. Player-to-player trading. NPC shop with programmatic sale events.

MARKET.PY · DB.PY
🧪
ALCHEMY

ALCHEMY_RECIPES catalogue with reagent requirements. apply_enchant() mutates items with stat bonuses. available_recipes() filters by player ingredients.

SKILLS_ALCHEMY.PY
🙏
FAITH SYSTEM

Full pantheon of gods, each with domains and day/night restrictions. Praying grants +10 favour. Blessings and curses applied as status effects.

FAITH.PY · LORE.PY
🤝
SOCIAL & CLANS

Friend requests, accept/decline flow. Clan creation with 24-char name limit, vault deposit, clan-wide buffs. Nation hierarchy above clans for politics.

SOCIAL.PY · DB.PY

Turn-based combat with status effects, skills, mana, cooldowns, and lore drops on mob defeat. Every fight tick is a pure function: given state → return new state. Night mobs gain +12 ATK / +6 DEF.

06

PLUGIN REGISTRY

All handlers live in bot/plugins/. Auto-loaded on boot via importlib scanner.

▶️
START
Character creation, path selection (Warrior/Rogue/Mage), profile display with all stats, world phase banner.
⚔️
COMBAT
Turn-based PvE fights. /fight command. Skill use, status effect resolution, loot and lore drops on defeat.
🏰
DUNGEON
Procedural floor generator. Room types: empty, mob, chest, trap, shrine, boss. Hidden dungeon rare spawns.
🎯
QUESTS
Daily quests from 30-quest pool (level-filtered). 10-chapter story arc. Quest progress tracking in DB.
🛒
MARKET
Player-to-player trading. NPC shop listings. Buy/sell/list commands. Tax sink on transactions.
🏆
LEADERBOARD
Global rankings by XP, gold, kills, dungeon clears, and faith score. Paginated output.
🙏
FAITH
/pray command with god pantheon keyboard. +10 faith favour per prayer. Day/night god restrictions. NPC interactions with rumours.
⚔️
PVP
/duel @target command. Inline accept/decline. Simulated combat up to 30 turns. XP and gold rewards. log_pvp_result() to DB.
🤝
SOCIAL
Friend requests, accept/decline flow. /clan create with 24-char cap. Clan vault gold deposits. Friend list display.
🎮
ARCADE
Slots (8 symbols, weighted), dice duel, coin flip. Bet 10–5000 gold. Triple 💎 pays 50×. Two-of-a-kind breaks even.
🧪
SKILLS & ALCHEMY
Brew potions from reagents using ALCHEMY_RECIPES. apply_enchant() upgrades gear stats. available_recipes() filters by inventory.
🎒
INVENTORY
/inventory command. View 4-slot chests. Equip/unequip items. rolled_stats() shows final stat bonuses from enchants.
📖
LORE
Lore fragment viewer. random_rumour(), random_ambient() surface world flavour. roll_lore_drop() on mob kill.

LORE & WORLD

In the age before memory, the God-King Solarius forged five kingdoms from chaos and bound them with the Eternal Covenant. The Sundering tore it apart. His divine essence shattered into Anima Cores. You are one who touched a Core. A Rogue. And Aethermoor is running out of time.
🏚️
VAELTHORN

The Shattered Hometown. A market town at the edge of Thornwood, half-ruined by the Sundering. Levels 1–10.

STARTER REGION
🌲
THORNWOOD

A cursed ancient forest haunted by bandits, vine horrors, and forest witches. Levels 8–20.

MID REGION
⛰️
ASHPEAK

Volcanic highland of the dwarven ruins. Home to automatons, trolls, and lava sprites. Levels 18–35.

HIGH REGION
🌑
VOIDMARSH

The edge of known reality. Void crawlers and shade lords stalk the dark. Night mobs amplified. Levels 30+.

ENDGAME REGION
☀️
SOLARIUM

The sealed pinnacle. Ancient dragons and void heralds guard the truth of the Sundering. Levels 40–50.

FINAL REGION
⚡
GODS & FAITH

Full pantheon with domains, day/night restrictions, boons, and curses. Faith stat affects defence formula. Praying costs no gold.
